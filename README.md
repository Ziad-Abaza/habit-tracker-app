# Habit Tracker

A modern, cross-platform mobile application built with Flutter to help you build and track your daily habits and routines. Stay consistent with your goals and develop positive lifestyle changes through intuitive tracking and reminders.

### Prerequisites

- Flutter SDK (latest stable version)
- Android Studio / Xcode (for emulator/simulator)
- VS Code or Android Studio with Flutter plugins

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/habit-tracker.git
   cd habit-tracker
   ```

   ```bash
   flutter pub get
   ```

3. Run the app:
